# Chess System Java ♟️

A robust terminal-based Chess game developed in Java. This project focuses on the core principles of **Object-Oriented Programming (OOP)**, providing a full game experience directly in the console.

## 📋 Features
* **Complete Rules**: Includes all standard chess moves.
* **Validation**: Real-time move validation and check/checkmate detection.

## 🛠️ Technologies
* **Language**: Java.
* **Architecture**: Domain-driven layers (Board, Pieces, Chess Match).
* **Version Control**: Git.

## 🚀 Getting Started

### Prerequisites
* **Java JDK** (version 11 or higher recommended).
* A terminal/console that supports **ANSI colors** (for the best visual experience).

### How to Run
1. Clone the repository:
   ```
   git clone [https://github.com/vitor-orasio/chess-system-java.git](https://github.com/vitor-orasio/chess-system-java.git)
   ```
2. Navigate to the project folder:
   ```
   cd chess-system-java
   ```
3. Compile the project:
   ```
   javac bin/application/Program.java
   ```
4. Run the application:
   ```
   java bin/application/Program
   ```
## 📐 Project Structure
* The system is divided into two main layers:

* Board Layer: Handles the board, positions, and generic pieces.

* Chess Layer: Contains the specific rules, piece movements, and match logic.
